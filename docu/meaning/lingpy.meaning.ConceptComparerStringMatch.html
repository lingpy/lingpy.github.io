lingpy.meaning.ConceptComparerStringMatch¶

class lingpy.meaning.ConceptComparerStringMatch(complete=True)¶

Implementation of a concept comparer based on a stemmer for spanish.

Parameters:None. :

See also

ConceptComparerBase, ConceptGraph

Notes

This is a sub-class of ConceptComparerBase. It uses a simple match of the stem of a given (spanish) string against a given context (that is supposed to be a stemmed spanish word stem).

Methods

compare_to_concept(element, concept) Compares a given element to a concept.
